## 📡 Wireless Network Basics

- **Definition** → Communication using radio waves (no cables).
    
- **Devices** → PCs, smartphones, IoT gadgets.
    
- **Uses** → Internet access, file sharing, services.
    

**Advantages**

- Mobility → Move freely within coverage.
    
- Easy installation → No cabling needed.
    
- Scalable → Add devices easily.
    

**Disadvantages**

- Interference → Walls, electronics, weather.
    
- Security risks → Easier to intercept if unsecured.
    
- Speed limits → Slower than wired.
    

## 📶 Wireless Router

- **Functions** →
    
    - Routing → Directs data to correct destination.
        
    - Wireless Access Point → Provides Wi-Fi coverage.
        
- **Components** →
    
    - WAN Port → Connects to ISP modem.
        
    - LAN Ports → Wired local devices.
        
    - Antennae → Transmit/receive signals.
        
    - Processor & Memory → Handle routing tasks.
        

**Example** → Home router connects smartphones, laptops, smart TVs → Internet via ISP modem.

## 📱 Mobile Hotspot

- **Definition** → Smartphone shares cellular data via Wi-Fi.
    
- **Range** → Few meters.
    
- **Drawbacks** → Battery drain, limited coverage.
    
- **Security** → Password-protected.
    
- **Example** → Traveling → Phone hotspot → Laptop connects to browse.
    

## 🗼 Cell Towers

- **Definition** → Structures with antennas covering geographic areas (cells).
    
- **Function** → Transmit/receive signals, managed by Base Station Controllers (BSC).
    
- **Backhaul** → Fiber/microwave links connect towers to core network.
    
- **Types** →
    
    - Macro cells → Large towers, rural coverage.
        
    - Micro/small cells → Urban, dense areas.
        
- **Example** → Road trip → Phone switches towers while streaming music.
    

## 📊 Frequencies in Wireless

- **2.4 GHz** → Older Wi-Fi, longer range, more interference.
    
- **5 GHz** → Newer Wi-Fi, faster, shorter range.
    
- **Cellular Bands** → 4G/5G → 700 MHz (long range) to 28 GHz+ (high-speed, short range).
    

**Trade-off** →

- Lower frequency → Longer range, less data.
    
- Higher frequency → Faster, shorter range.
    

## 🖼️ Flow Diagram (Wireless Communication)

Code

```
Devices (PC, Phone, IoT)
        |
   Wireless Router (Wi-Fi)
        |
   ISP Modem → Internet
        |
   Cell Towers (4G/5G coverage)
        |
   Mobile Hotspot (personal sharing)
```

## 🎯 Key Takeaway

- **Wi-Fi** → Local wireless access.
    
- **Cell Towers** → Wide-area coverage.
    
- **Hotspot** → Personal data sharing.
    
- **Frequencies** → Balance between range & speed.

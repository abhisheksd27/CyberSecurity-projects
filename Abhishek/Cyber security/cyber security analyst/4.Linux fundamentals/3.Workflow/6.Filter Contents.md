# 🛠 Filtering Tools in Linux

## 🔑 Why Filtering Matters

When working with large files (like `/etc/passwd`), raw output can be overwhelming. Filtering tools help us **search, sort, extract, and format** data so we only see what we need.

## 📌 Core Tools

### 1. **sort**

- **Purpose**: Arrange lines alphabetically or numerically.
    
- **Example**:
    
    bash
    
    ```
    cat /etc/passwd | sort
    ```
    
- **Result**: Output starts with `_apt` instead of `root`.
    

### 2. **grep**

- **Purpose**: Search for patterns in text.
    
- **Example (find bash users)**:
    
    bash
    
    ```
    cat /etc/passwd | grep "/bin/bash"
    ```
    
- **Exclude results**:
    
    bash
    
    ```
    grep -v "false\|nologin"
    ```
    

### 3. **cut**

- **Purpose**: Extract specific fields using delimiters.
    
- **Example (get usernames only)**:
    
    bash
    
    ```
    cut -d":" -f1 /etc/passwd
    ```
    

### 4. **tr**

- **Purpose**: Replace or delete characters.
    
- **Example (replace** `:` **with spaces)**:
    
    bash
    
    ```
    tr ":" " "
    ```
    

### 5. **column**

- **Purpose**: Format output into neat tables.
    
- **Example**:
    
    bash
    
    ```
    ... | column -t
    ```
    

### 6. **awk**

- **Purpose**: Process text using fields (`$1`, `$NF`).
    
- **Example (show username + shell)**:
    
    bash
    
    ```
    awk '{print $1, $NF}'
    ```
    

### 7. **sed**

- **Purpose**: Stream editor for substitutions.
    
- **Example (replace** `bin` **with** `HTB`**)**:
    
    bash
    
    ```
    sed 's/bin/HTB/g'
    ```
    

### 8. **wc**

- **Purpose**: Count lines, words, or characters.
    
- **Example (count lines)**:
    
    bash
    
    ```
    wc -l
    ```
    

## 🔄 Flow Diagram

Here’s how these tools can be chained together:

Code

```
cat /etc/passwd
        │
        ▼
      grep (filter patterns)
        │
        ▼
      cut (extract fields)
        │
        ▼
      tr (replace chars)
        │
        ▼
      column (tabular view)
        │
        ▼
      awk (custom fields)
        │
        ▼
      sed (substitute text)
        │
        ▼
      wc (count results)
```

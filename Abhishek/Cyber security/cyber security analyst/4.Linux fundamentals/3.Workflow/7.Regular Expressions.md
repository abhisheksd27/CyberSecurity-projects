## 🌟 What is RegEx?

- A **pattern-matching language** used to search, filter, and manipulate text.
    
- Works in many tools (like `grep`, `sed`, `awk`) and programming languages.
    
- Uses **metacharacters** (special symbols) to define search rules.
    

## 📌 Grouping Operators

|Operator|Description|Example|
|---|---|---|
|`(a)`|Round brackets group patterns|`(my|false)` → matches "my" OR "false"|
|`[a-z]`|Square brackets define character classes|`[0-9]` → matches any digit|
|`{1,10}`|Curly brackets define quantifiers|`a{3}` → matches "aaa"|
|`|`|OR operator|`(cat|dog)` → matches "cat" or "dog"|
|`.*`|Acts like AND (sequence)|`(my.*false)` → "my" followed by "false"|

## ⚙️ Examples

### OR Operator

bash

```
grep -E "(my|false)" /etc/passwd
```

Finds lines containing **either** "my" or "false".

### AND Operator

bash

```
grep -E "(my.*false)" /etc/passwd
```

Finds lines containing **both** "my" and "false" in order.

### Equivalent AND using two greps

bash

```
grep -E "my" /etc/passwd | grep -E "false"
```

## 📝 Practice Tasks (using `/etc/ssh/sshd_config`)

1. **Show all lines that do not contain** `#`**:**
    
    bash
    
    ```
    grep -v "#" /etc/ssh/sshd_config
    ```
    
2. **Search for all lines starting with** `Permit`**:**
    
    bash
    
    ```
    grep -E "^Permit" /etc/ssh/sshd_config
    ```
    
3. **Search for all lines ending with** `Authentication`**:**
    
    bash
    
    ```
    grep -E "Authentication$" /etc/ssh/sshd_config
    ```
    
4. **Search for all lines containing the word** `Key`**:**
    
    bash
    
    ```
    grep "Key" /etc/ssh/sshd_config
    ```
    
5. **Search for all lines beginning with** `Password` **and containing** `yes`**:**
    
    bash
    
    ```
    grep -E "^Password.*yes" /etc/ssh/sshd_config
    ```
    
6. **Search for all lines ending with** `yes`**:**
    
    bash
    
    ```
    grep -E "yes$" /etc/ssh/sshd_config
    ```

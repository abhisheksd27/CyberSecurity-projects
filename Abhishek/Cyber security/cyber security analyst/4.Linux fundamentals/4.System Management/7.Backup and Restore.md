## 📘 Concept

- Protects data from loss/corruption.
    
- Tools: **Rsync, Duplicity, Deja Dup**.
    
- Encryption ensures sensitive data is secure.
    

**Analogy:**

- Rsync → fast transport, only moves changes.
    
- Duplicity → secure vault with encryption.
    
- Deja Dup → simple safe with GUI.
    

## 🔧 Rsync

### Install

bash

```
sudo apt install rsync -y
```

### Backup Directory → Remote Server

bash

```
rsync -av /path/to/mydirectory user@backup_server:/path/to/backup/directory
```

- `-a` → preserve attributes.
    
- `-v` → verbose output.
    

### Advanced Backup (compression + incremental)

bash

```
rsync -avz --backup --backup-dir=/path/to/backup/folder --delete \
/path/to/mydirectory user@backup_server:/path/to/backup/directory
```

- `-z` → compression.
    
- `--backup` → incremental backups.
    
- `--delete` → remove files not in source.
    

### Restore Backup

bash

```
rsync -av user@remote_host:/path/to/backup/directory /path/to/mydirectory
```

## 🔒 Secure Rsync (via SSH)

bash

```
rsync -avz -e ssh /path/to/mydirectory user@backup_server:/path/to/backup/directory
```

- Encrypts transfer → confidentiality + integrity.
    

## 🔄 Auto-Synchronization (Rsync + Cron)

### Steps:

1. Generate SSH key pair:
    

bash

```
ssh-keygen -t rsa -b 2048
ssh-copy-id user@backup_server
```

2. Create script `RSYNC_Backup.sh`:
    

bash

```
#!/bin/bash
rsync -avz -e ssh /path/to/mydirectory user@backup_server:/path/to/backup/directory
```

3. Make executable:
    

bash

```
chmod +x RSYNC_Backup.sh
```

4. Schedule with cron:
    

bash

```
crontab -e
```

Add entry:

Code

```
0 * * * * /path/to/RSYNC_Backup.sh
```

→ runs every hour at minute 0.

## 🔐 Other Tools

- **Duplicity** → Rsync + encryption (supports cloud storage).
    
- **Deja Dup** → GUI, simple, supports encrypted backups.
    
- Extra encryption: **GnuPG, eCryptfs, LUKS**.
    

## 🧠 Key Takeaways

- Rsync = efficient, incremental, secure with SSH.
    
- Duplicity = encrypted backups, cloud-ready.
    
- Deja Dup = user-friendly GUI.

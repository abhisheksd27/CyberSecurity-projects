# 📂 Important Windows Directories

## 📁 1. PerfLogs

- Stores **performance logs**
- Usually empty by default

👉 Example:

- CPU usage logs

---

## 📁 2. Program Files

- Stores installed applications

👉 Rules:

- 64-bit OS → only **64-bit apps**
- 32-bit OS → all apps

---

## 📁 3. Program Files (x86)

- Exists only in **64-bit Windows**
- Stores:
    - 32-bit apps
    - 16-bit apps

👉 Example:

- Old software compatibility

---

## 📁 4. ProgramData

- **Hidden folder**
- Stores **application data shared across users**

👉 Example:

- Software settings used by all users

---

6

- Contains **user accounts**

### Important Subfolders:

### 🔹 Default

- Template for new users

👉 When a new user is created → copied from here

---

### 🔹 Public

- Shared files for all users
- Accessible over network

---

### 🔹 AppData (Hidden)

- Stores user-specific app data

#### Subfolders:

- **Roaming**
    - Moves with user profile
    - Example: browser settings
- **Local**
    - Only for that PC
    - Not shared
- **LocalLow**
    - Low security level
    - Used by browsers (safe mode)

---

## 📁 6. Windows

- Contains **core OS files**

👉 Example:

- System executables
- Updates

---

## 📁 7. System / System32 / SysWOW64

## 📌 Purpose:

- Stores **DLL files (libraries)**
- Core Windows functionality

👉 Important:

- When a program runs → it loads files from here

### 🔹 System32

- Main system files (64-bit)

### 🔹 SysWOW64

- 32-bit compatibility on 64-bit systems

---

## 📁 8. WinSxS

- **Windows Component Store**

👉 Stores:

- Updates
- Backup files
- System components

👉 Important:

- Used for system recovery

---

# 💻 Exploring Directories (Command Line)

## 📌 1. dir Command

### Syntax:

dir C:\ /a

### 📌 What it does:

- Lists all files & folders
- `/a` → shows hidden files

👉 Example Output:

Program Files  
Users  
Windows  
pagefile.sys

# 🧠 Pentester Important Notes

- 🔍 **Program Files** → check installed software
- 🔍 **Users\AppData** → sensitive data (tokens, configs)
- 🔍 **System32** → critical system files
- 🔍 **Public folder** → possible data leakage
- 🔍 Hidden folders → often contain secrets

---

# 🧾 Final Summary (Quick Revision)

- Root directory = `C:\`
- Key folders:
    - Program Files → apps
    - Users → user data
    - Windows → OS files
    - System32 → core system
- Commands:
    - `dir` → list files
    - `tree` → view structure

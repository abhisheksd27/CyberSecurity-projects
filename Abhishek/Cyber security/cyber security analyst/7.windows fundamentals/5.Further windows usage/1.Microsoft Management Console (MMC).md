# 🔷 1. What is MMC?

👉 **Microsoft Management Console (MMC)** is a framework that lets you manage Windows using **administrative tools (snap-ins)**.

📌 Think of it as:

"A container where you plug in admin tools"

---

# 🧩 2. Key Concept: Snap-ins

👉 MMC itself does **nothing alone** — it becomes powerful through **snap-ins**

### 🔹 What are Snap-ins?

- Small tools used to manage specific parts of Windows
- Can manage:
    - Users
    - Services
    - Disks
    - Devices
    - Security policies

---

## 🔹 Common Snap-ins You Should Know

|Snap-in|Purpose|
|---|---|
|Services|Manage background services|
|Event Viewer|View logs|
|Device Manager|Manage hardware|
|Disk Management|Manage disks/partitions|
|Local Users & Groups|Manage users|

---

# ⚙️ 3. How MMC Works

### Flow:

MMC Console  
   ↓  
Snap-ins (tools)  
   ↓  
System Components

---

# 💻 4. How to Open MMC

### 🔹 Method:

mmc

👉 Opens a **blank console**

---

# 🛠️ 5. Adding Snap-ins (Important)

### Steps:

1. Open MMC
2. Go to:
    
    File → Add/Remove Snap-ins
    
3. Select tools
4. Choose:
    - Local computer ✅
    - Remote computer 🌐

---

# 🌐 6. Local vs Remote Management

### 🔹 Local

- Manage your own system

### 🔹 Remote

- Manage another system over network

👉 Example:

- Manage services on another PC

---

# 💾 7. Saving MMC Configuration

👉 You can save your setup as:

.msc file

### 🔹 Why useful?

- Reuse tools setup
- Share with admins
- Quick access

📌 Example:

- `services.msc`
- `eventvwr.msc`

# 🔐 9. Security Perspective (VERY IMPORTANT 🔥)

### ✔ Useful for:

- Monitoring suspicious activity
- Checking logs (Event Viewer)

### ❗ Risk:

- If attacker gets access:
    - Can disable services
    - Add users
    - Modify system settings

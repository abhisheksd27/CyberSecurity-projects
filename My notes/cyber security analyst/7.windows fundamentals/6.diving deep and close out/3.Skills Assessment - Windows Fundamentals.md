## ✅ **Step 8: Use PowerShell to List Service Details**

Open PowerShell:

---

### 🔹 Command 1 (Basic):

Get-Service | Select-Object -First 5

---

### 🔹 Command 2 (Detailed):

Get-Service -Name wuauserv | Format-List *

👉 Example service:

- `wuauserv` = Windows Update

---

### 🔹 Command 3 (Only Running Services):

Get-Service | Where-Object {$_.Status -eq "Running"}

---

## 🧠 WHAT THIS SHOWS

You can:

- Detect suspicious services
- Check if malware modified services
- Monitor system behavior

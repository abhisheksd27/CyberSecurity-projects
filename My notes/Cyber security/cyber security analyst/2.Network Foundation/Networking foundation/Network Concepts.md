## 📚 OSI Model (7 Layers)

1. **Physical** → Raw bits, cables, hubs.
    
2. **Data Link** → Node-to-node, MAC addresses, switches.
    
3. **Network** → Routing, IP addresses, routers.
    
4. **Transport** → TCP (reliable), UDP (fast).
    
5. **Session** → Establish/maintain sessions, APIs.
    
6. **Presentation** → Translation, encryption, compression.
    
7. **Application** → User-facing services (HTTP, FTP, SMTP, DNS).
    

**Flow Example (File Transfer):** Application → Presentation → Session → Transport → Network → Data Link → Physical

## 🌐 TCP/IP Model (4 Layers)

1. **Link Layer** → Ethernet, Wi-Fi (physical + data link).
    
2. **Internet Layer** → IP addressing, routing, ICMP.
    
3. **Transport Layer** → TCP/UDP for communication.
    
4. **Application Layer** → HTTP, FTP, SMTP (web, file, email).
    

**Flow Example (Accessing Website):** Application (HTTP) → Transport (TCP) → Internet (IP routing) → Link (Ethernet/Wi-Fi)

  ## 🔑 Common Protocols

- **HTTP** → Web browsing (Application).
    
- **FTP** → File transfers (Application).
    
- **SMTP** → Email (Application).
    
- **TCP** → Reliable, ordered delivery (Transport).
    
- **UDP** → Fast, connectionless (Transport).
    
- **IP** → Routing packets (Internet).
    

## 📡 Transmission

- **Types** → Analog (continuous), Digital (bits).
    
- **Modes** →
    
    - Simplex → One-way only (keyboard → PC).
        
    - Half-duplex → Two-way, not simultaneous (walkie-talkie).
        
    - Full-duplex → Two-way simultaneous (telephone).
        
- **Media** →
    
    - Wired → Twisted pair, coaxial, fiber optics.
        
    - Wireless → Radio (Wi-Fi), microwaves (satellite), infrared (short-range).

## 🎯 Key Takeaway

- **OSI** → 7 layers, theoretical framework.
    
- **TCP/IP** → 4 layers, practical internet backbone.
    
- **Protocols** → Rules for communication.
    
- **Transmission** → How data travels (types, modes, media).

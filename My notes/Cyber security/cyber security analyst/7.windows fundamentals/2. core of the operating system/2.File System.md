# 📁 FAT32 (File Allocation Table)

## 📌 What is FAT32?

- Uses **32-bit addressing** for storage
- Common in:
    - USB drives
    - SD cards

---

## ✅ Advantages

- Works on:
    - Windows
    - macOS
    - Linux
- Supported by:
    - Cameras 📷
    - Consoles 🎮
    - Phones 📱

---

## ❌ Disadvantages

- Max file size = **4GB**
- No:
    - Encryption 🔒
    - Permissions
    - Compression

---

## 📌 Example

- Pendrive formatted in FAT32
- Cannot copy a **5GB movie file** ❌

# 🔐 NTFS Permissions

## 📌 What are Permissions?

- Control **who can access files/folders**

---

## 📊 Types of Permissions

|Permission|Meaning|
|---|---|
|Full Control|Read, write, delete|
|Modify|Read, write, delete|
|Read & Execute|View + run files|
|Read|Only view|
|Write|Add/edit files|
|List Folder|View folder|
|Traverse|Move through folders|

---

## 📌 Example

- User A:
    - Can read file ✅
    - Cannot delete ❌

---

# 🔁 Permission Inheritance

- Child files inherit from parent folder

👉 Example:

C:\Users\Abhi\Documents

- If **Documents = Read only**  
    → All files inside = Read only

👉 Admin can:

- Disable inheritance
- Set custom permissions

---

# 🧰 icacls Command (Important)

## 📌 What is icacls?

- Command-line tool to:
    - View permissions
    - Modify permissions

---

## 📌 View Permissions

icacls C:\Windows

👉 Output Example:

SYSTEM:(F)  
Users:(RX)

---

## 📌 Permission Codes

|Code|Meaning|
|---|---|
|F|Full|
|M|Modify|
|RX|Read + Execute|
|R|Read|
|W|Write|

---

## 📌 Inheritance Codes

|Code|Meaning|
|---|---|
|(CI)|Folder inherit|
|(OI)|File inherit|
|(I)|Inherited|

---

## 📌 Grant Permission

icacls C:\Users /grant joe:F

👉 Gives:

- User **joe → Full access**

---

## 📌 Remove Permission

icacls C:\Users /remove joe

---

## ⚠️ Important Note

- Without `(OI)(CI)`:
    - Permission applies only to folder
    - Not subfolders/files

---

# 🧠 Pentester Important Points

- 🔍 Check:
    - Weak permissions
    - Writable folders
- 🔍 Misconfigurations:
    - Everyone = Full Control ❌
- 🔍 Look in:
    - `Users\AppData`
    - Shared folders

👉 Example attack:

- Modify executable → privilege escalation

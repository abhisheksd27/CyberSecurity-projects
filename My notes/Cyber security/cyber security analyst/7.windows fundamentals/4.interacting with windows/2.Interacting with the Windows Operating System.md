# 🔷 1. Big Picture: How Windows OS Works

Think of Windows as **layers working together**:

User (GUI / CMD / PowerShell)  
        ↓  
Processes & Services  
        ↓  
File System (NTFS, FAT32)  
        ↓  
Hardware

---

# 📁 2. File System (Storage Layer)

### 🔹 Types of File Systems

- FAT32 → Old, compatible, limited (4GB file max)
- exFAT → Modern removable storage
- **NTFS → Default & most important**

### 🔹 Why NTFS is Important

- Security (permissions)
- Large file support
- Journaling (logs changes)
- Stability

---

# 🔐 3. NTFS Permissions (Core Security)

### 🔹 Basic Permissions

- Full Control
- Modify
- Read & Execute
- Read
- Write

### 🔹 Key Concept

👉 **Inheritance**

- Child folders/files inherit permissions from parent

### 🔹 Command to Check Permissions

icacls C:\Windows

---

# 🌐 4. NTFS vs Share Permissions (VERY IMPORTANT 🔥)

|Feature|NTFS Permissions|Share Permissions|
|---|---|---|
|Applies to|Local system|Network (SMB)|
|Granularity|Very detailed|Basic|
|Works when|Local or remote|Only remote|

### 🔥 Final Access Rule:

Effective Permission = MOST RESTRICTIVE (NTFS vs Share)

---

# 📡 5. SMB (File Sharing)

- Protocol used for sharing files over network
- Works on port **445 / 139**
- Used in enterprises

### 🔹 Example Tool:

smbclient -L //IP

---

# 🔥 Security Note

- Vulnerability: EternalBlue
- Used in ransomware attacks (e.g., WannaCry)

---

# 🧩 6. Windows Services (Background Workers)

### 🔹 What are Services?

- Run in background
- Start automatically
- No user interaction needed

### 🔹 Examples:

- Windows Update
- Networking
- Authentication

### 🔹 Manage Services:

- GUI: `services.msc`
- CMD:

sc query

- PowerShell:

Get-Service

---

# ⚠️ 7. Service Misconfiguration (HUGE for Cybersecurity)

### 🔥 Attack Vectors:

- Weak permissions
- Replace executable path
- Run malicious program

### 🔹 Example:

sc config service binPath=malicious.exe

👉 This can lead to:

- Privilege escalation
- Persistence

---

# 🧠 8. Important Windows Processes

### 🔹 Critical Processes

- `lsass.exe` → Authentication (VERY IMPORTANT 🔥)
- `winlogon.exe` → Login handling
- `services.exe` → Service manager
- `svchost.exe` → Runs services

### 🔥 High-value target:

- LSASS → stores credentials in memory

---

# 🧑‍💻 9. Windows Sessions

### 🔹 Interactive

- User login (local, RDP)

### 🔹 Non-Interactive

- System accounts:
    - LocalSystem (highest privilege)
    - LocalService
    - NetworkService

# ⚙️ 11. PowerShell Concepts

### 🔹 Cmdlet Format

Verb-Noun

Examples:

- Get-Service
- Set-Location

# 🔒 12. Execution Policy (Security Feature)

### 🔹 Types:

- Restricted (no scripts)
- RemoteSigned (default)
- Bypass (no restrictions)

### 🔹 Change Policy:

Set-ExecutionPolicy Bypass -Scope Process

# 🧠 FINAL UNDERSTANDING (IMPORTANT 🔥)

### ✔ Windows Security depends on:

- NTFS permissions
- Share permissions
- Service configurations
- Running processes
- User sessions

### ✔ Most Attacks Target:

- Misconfigured services
- Weak permissions
- SMB shares
- LSASS memory

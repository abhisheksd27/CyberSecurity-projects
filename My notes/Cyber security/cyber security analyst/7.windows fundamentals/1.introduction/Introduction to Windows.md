- **Windows** is an **Operating System (OS)** developed by Microsoft
- It allows users to interact with a computer using a **Graphical User Interface (GUI)**
- Used in:
    - Personal computers 💻
    - Servers 🖥️
    - Enterprise environments 🏢

👉 For **penetration testers**, Windows is important because:

- Most companies use Windows systems
- Many attacks target Windows vulnerabilities
- It can also be used as a **testing platform**

- Most real-world systems run **Windows or Linux**
- You must know:
    - 🔴 How to **attack** Windows (find vulnerabilities)

    - ⚙️ How to **use it for hacking tools**

# ⚙️ Getting System Info (PowerShell)

## Command:

Get-WmiObject -Class win32_OperatingSystem | select Version,BuildNumber

## 📌 What it does:

- Shows:
    - OS version
    - Build number

👉 Example Output:

Version    BuildNumber  
10.0.19041 19041

## 🧠 Other Useful Classes:

- `Win32_Process` → running processes
- `Win32_Service` → services
- `Win32_BIOS` → BIOS info

    - IP Address (to find system)

- Network = Street 🏠
- IP Address = House
- Port = Door 🚪

## 🛠️ RDP Tools

### 1️⃣ On Windows:

- Built-in tool:
    - **Remote Desktop Connection (mstsc.exe)**

### 2️⃣ On Linux:

- Tool:
    - **xfreerdp**

👉 Example:

xfreerdp /u:username /p:password /v:IP

---

## ⚠️ Important Notes

- RDP is **disabled by default**
- Must be enabled manually
- Pentesters look for:
    - Saved `.rdp` files
    - Weak credentials

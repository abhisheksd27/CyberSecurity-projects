- **Services = background programs**
- Run **continuously** without user interaction

👉 Key features:

- Start automatically at **boot**
- Run even after **user logs out**
- Perform critical system tasks

---

## 📌 Examples of Service Tasks

- Networking 🌐
- Windows updates 🔄
- User authentication 🔐
- System monitoring 📊

---

## 🧠 Service Control Manager (SCM)

- Manages all services
- Open using:

services.msc

👉 Shows:

- Name
- Status
- Startup type
- User account

---

## 💻 Command Line (PowerShell)

### List running services:

Get-Service | ? {$_.Status -eq "Running"}

---

## 📊 Service States

|State|Meaning|
|---|---|
|Running|Active|
|Stopped|Not running|
|Paused|Temporarily stopped|

---

## ⚙️ Startup Types

|Type|Meaning|
|---|---|
|Automatic|Starts at boot|
|Manual|Starts when needed|
|Disabled|Cannot start|

---

## 📂 Types of Services

- Local Services
- Network Services
- System Services

### 📌 What is LSASS?

- Full form:  
    **Local Security Authority Subsystem Service**

### 📌 Role:

- Authenticates users
- Creates access tokens
- Handles passwords

---

## ⚠️ Why Hackers Target LSASS

- Stores:
    - Passwords
    - Hashes
- Tools like:
    - Mimikatz extract credentials

👉 Very high-value target 🎯

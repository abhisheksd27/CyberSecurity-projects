## 📚 Core Networking Protocols

- **WEP / WPA / TKIP** → Wireless security protocols (WEP = outdated, WPA = stronger, TKIP = weaker).
    
- **SSH** → Secure remote login & command execution.
    
- **FTP** → File transfer between systems.
    
- **SMTP** → Email sending/receiving.
    
- **HTTP** → Web data transfer (client ↔ server).
    
- **SMB / NFS** → File sharing protocols (Windows = SMB, Unix/Linux = NFS).
    
- **SNMP** → Manage/monitor network devices.
    
- **NTP** → Synchronize system clocks.
    

## 🌍 Routing & Network Segmentation

- **VLAN / VTP** → Logical segmentation of networks across switches.
    
- **RIP** → Distance-vector routing (basic).
    
- **OSPF** → Link-state routing (efficient, scalable).
    
- **IGRP / EIGRP** → Cisco proprietary routing protocols.
    
- **NAT** → Translate private IPs → public IP (Internet access).
    

## 🔐 Security & Authentication

- **PGP** → Encrypt emails/files.
    
- **IPsec** → Secure encrypted communication (VPNs).
    
- **IKE / GRE / PPTP** → VPN tunneling & encryption protocols.
    
- **EAP / LEAP / PEAP** → Authentication frameworks for wireless networks.
    
- **TACACS** → Centralized authentication/authorization/accounting.
    

## 📡 Redundancy & Loop Prevention

- **HSRP / VRRP** → Router redundancy (failover).
    
- **STP** → Prevents Layer 2 loops in Ethernet networks.
    

## 📞 Communication Protocols

- **SIP** → Signaling for voice/video sessions.
    
- **VoIP** → Voice calls over IP networks.
    
- **NNTP** → Usenet/newsgroup messaging.
    
- **CDP** → Cisco device discovery protocol.
    

## 🛠️ Management & Systems

- **SMS (Systems Management Server)** → Manage devices/networks.
    
- **MBSA** → Microsoft tool for detecting vulnerabilities.
    
- **SCADA** → Industrial control systems (manufacturing, utilities).
    

## 🌐 Web & Application Terms

- **AJAX** → Dynamic web pages (JavaScript + XML/JSON).
    
- **ISAPI** → Web server extensions (performance-oriented).
    
- **URI / URL** → Resource identifiers (URL = specific location).
    
- **CRLF** → End-of-line marker in text files.
    

## 🖼️ Quick Mental Map

Code

```
Security → WEP, WPA, IPsec, PGP, EAP
Routing → RIP, OSPF, EIGRP, NAT
Redundancy → HSRP, VRRP, STP
File Sharing → SMB, NFS
Management → SNMP, SMS, MBSA, SCADA
Communication → SIP, VoIP, NNTP
Web → HTTP, AJAX, ISAPI, URI/URL
```

## 🎯 Key Takeaway

- These terms form the **alphabet of networking**.
    
- Focus on **security (IPsec, EAP, VPNs)**, **routing (OSPF, NAT)**, and **redundancy (HSRP, STP)** — they’re most critical for cybersecurity roles.
    
- Expand this list gradually as you encounter new protocols in labs or certifications.

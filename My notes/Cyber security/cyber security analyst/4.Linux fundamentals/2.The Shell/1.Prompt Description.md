## 🖥️ Bash Prompt Basics

- **Default format**:
    
    Code
    
    ```
    <username>@<hostname><current working directory>$
    ```
    
- **Home directory** → shown as `~`.
    
- **Symbols**:
    
    - `$` → normal user.
        
    - `#` → root (privileged user).
        

**Examples**:

Code

```
user@htb[~]$   → logged in as user, home directory
root@htb[/htb]# → logged in as root, system directory
```

## ⚙️ PS1 Variable

- **Controls how the prompt looks**.
    
- Acts like a **template** for the text shown before the cursor.
    
- Can include username, hostname, directory, time, or even colors.
    
- Defined in `.bashrc` (user shell config file).
    

## 🔑 Special Characters for Customization

| Character | Meaning                               |
| --------- | ------------------------------------- |
| `\u`      | Current username                      |
| `\h`      | Hostname                              |
| `\w`      | Current working directory (full path) |
| `\d`      | Date (e.g., Mon Feb 6)                |
| `\t`      | Current time (HH:MM:SS, 24h)          |
| `\@`      | Current time (12h with AM/PM)         |
| `\H`      | Full hostname                         |
| `\j`      | Number of jobs                        |
| `\s`      | Shell name                            |
| `\n`      | Newline                               |
| `\r`      | Carriage return                       |

## 🛠️ Example Custom Prompts

- **Basic**:
    
    Code
    
    ```
    PS1="\u@\h:\w$ "
    → user@host:/current/path$
    ```
    
- **With date/time**:
    
    Code
    
    ```
    PS1="[\d \t] \u@\h:\w$ "
    → [Tue Mar 10 03:53:00] abhishek@htb:/home$
    ```
    
- **Colorized (using ANSI codes)**:
    
    Code
    
    ```
    PS1="
    
    ```
    

\e[32m

\u@\h:\w$

\e[0m

" → username@hostname:/path$ (in green)

## 📂 Useful for Cybersecurity
- **Penetration testing** → customize prompt to show target IP, current directory, or last command status.  
- **Documentation** → combine with `.bash_history` or `script` to log commands with timestamps.  
- **Efficiency** → instantly know if you’re root, which system you’re on, and where you are in the file system.

### 📝 Nano Editor

|Command|Description|
|---|---|
|`nano <filename>`|Opens (or creates) a file in the **Nano text editor**|
|`cat <filename>`|**Displays** the contents of a file in the terminal|

#### Nano Keyboard Shortcuts

| Shortcut                    | Description                                      |
| --------------------------- | ------------------------------------------------ |
| `[CTRL + O]`                | **Save** (Write Out) the file                    |
| `[CTRL + X]`                | **Exit** the Nano editor                         |
| `[CTRL + W]`                | **Search** for a word or phrase                  |
| `[CTRL + W]` then `[ENTER]` | Jump to the **next match** in search             |
| `[CTRL + K]`                | **Cut** a line of text                           |
| `[CTRL + U]`                | **Uncut/Paste** text                             |
| `[CTRL + C]`                | Show **cursor position** / Cancel current action |
| `[CTRL + G]`                | Open **help** menu                               |

> The `^` symbol in Nano's interface represents the `[CTRL]` key.

### ⚡ Vim Editor

|Command|Description|
|---|---|
|`vim`|Opens the **Vim editor**|
|`vim <filename>`|Opens or creates a specific file in Vim|
|`vimtutor`|Launches the built-in **Vim tutorial** (takes ~25–30 mins)|

#### Vim's 6 Modes

|Mode|Description|
|---|---|
|**Normal**|Default mode on startup — all keystrokes are **editor commands**, not text input|
|**Insert**|**Type and insert text** into the file buffer|
|**Visual**|**Highlight/select** a section of text to copy, delete, or replace|
|**Command**|Enter **single-line commands** at the bottom (e.g., save, quit, replace)|
|**Replace**|**Overwrites** existing characters with newly typed ones|
|**Ex**|Run **multiple sequential commands** without returning to Normal mode|

#### Essential Vim Commands

|Command|Description|
|---|---|
|`:q`|**Quit** Vim|
|`:q!`|**Force quit** without saving|
|`:w`|**Save** the file|
|`:wq`|**Save and quit**|
|`i`|Enter **Insert mode**|
|`ESC`|Return to **Normal mode**|
|`:Tutor`|Open **vimtutor** from within Vim|

### 🔐 Important Linux Files (Security Context)

| File          | Description                                                                                               |
| ------------- | --------------------------------------------------------------------------------------------------------- |
| `/etc/passwd` | Contains user info: **usernames, UIDs, GIDs, home directories** — historically stored password hashes too |
| `/etc/shadow` | Stores **password hashes** with stricter permissions — replaced passwd for storing credentials            |
|               |                                                                                                           |

> Misconfigured permissions on these files can expose sensitive data or lead to **privilege escalation** — a key focus area in penetration testing.

### 💡 Nano vs Vim — Quick Comparison

|Feature|Nano|Vim|
|---|---|---|
|Difficulty|**Beginner-friendly**|**Steep learning curve**|
|Interface|Simple, menu-driven|Modal, command-driven|
|Speed|Moderate|Extremely fast once mastered|
|Extensibility|Limited|Highly extensible (integrates grep, awk, sed, etc.)|
|Best for|Quick edits|Power users, complex editing|

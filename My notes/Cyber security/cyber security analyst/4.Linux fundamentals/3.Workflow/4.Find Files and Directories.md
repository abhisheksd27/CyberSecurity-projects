### 🔍 Finding Files & Directories — 3 Main Tools

---

### 1. `which` — Find Program Location

|Command|Description|
|---|---|
|`which <program>`|Returns the **full path** of an executable/program if it exists on the system|

**Example:**

bash

```bash
which python     # returns /usr/bin/python
which curl
which wget
which gcc
```

> If the program doesn't exist, **no output** is returned.

---

### 2. `find` — Powerful File Search with Filters

**Syntax:**

bash

```bash
find <location> <options>
```

#### `find` Options & Filters

|Option|Description|
|---|---|
|`-type f`|Search for **files** only (`f` = file, `d` = directory)|
|`-name *.conf`|Filter by **filename** — `*` is a wildcard meaning "any"|
|`-user root`|Filter by **file owner**|
|`-size +20k`|Filter by **size** — `+20k` means larger than 20 KiB|
|`-newermt 2020-03-03`|Only show files **newer than** a specified date|
|`-exec ls -al {} \;`|**Execute a command** on each result — `{}` is a placeholder for each found file|
|`2>/dev/null`|Redirect **error messages** to null (suppress errors from showing in terminal)|

**Full Example:**

bash

```bash
find / -type f -name *.conf -user root -size +20k -newermt 2020-03-03 -exec ls -al {} \; 2>/dev/null
```

This finds all `.conf` files owned by root, larger than 20KB, newer than March 3 2020, and lists their details.

---

### 3. `locate` — Fast Search Using a Database

|Command|Description|
|---|---|
|`locate <pattern>`|Searches a **local database** for files — much faster than `find`|
|`sudo updatedb`|**Updates the local database** that `locate` uses to search|

**Example:**

bash

```bash
sudo updatedb        # refresh the database first
locate *.conf        # find all .conf files instantly
```

---

### ⚖️ `find` vs `locate` — Comparison

|Feature|`find`|`locate`|
|---|---|---|
|Speed|Slower (searches live filesystem)|**Much faster** (uses a database)|
|Filter options|**Many** (size, date, owner, type, etc.)|Limited|
|Up-to-date results|Always **current**|Depends on last `updatedb` run|
|Best for|Complex, filtered searches|Quick lookups by filename|

---

### 💡 Key Concepts

|Concept|Description|
|---|---|
|`*` wildcard|Matches **any characters** — e.g., `*.conf` matches all `.conf` files|
|`2>/dev/null`|Suppresses **STDERR** (error output) so only valid results are shown|
|`-exec {} \;`|Runs a command on **each result** found by `find` — `{}` is replaced by the file path|
|`updatedb`|Must be run as **sudo** — refreshes `locate`'s file index|

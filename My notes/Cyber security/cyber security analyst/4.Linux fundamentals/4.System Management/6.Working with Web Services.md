# 🌐 Working with Web Services (Linux)

## 📘 Concept

- Web servers enable communication between browsers ↔ servers.
    
- Popular: **Apache, Nginx, IIS**.
    
- **Apache** = most widely used, modular, customizable.
    
- Analogy: Apache = **house foundation**; modules = rooms/features.
    

## 🔧 Apache Web Server

### Features

- **Modularity** → extend with modules:
    
    - `mod_ssl` → encrypt communication.
        
    - `mod_proxy` → traffic redirection.
        
    - `mod_headers`, `mod_rewrite` → control headers & URLs.
        
- Supports **static & dynamic content** (PHP, Perl, Python, Ruby, etc.).
    

### Install & Start

bash

```
sudo apt install apache2 -y
sudo systemctl start apache2
```

- Default page: `http://localhost` → “It works!” message.
    

### Change Port

- Edit `/etc/apache2/ports.conf`:
    

apache

```
Listen 8080
```

- Restart Apache → access via `http://localhost:8080`.
    

### Verify with curl

bash

```
curl -I http://localhost:8080
```

**Output:** `HTTP/1.1 200 OK` → confirms server running.

## 🖥️ Command-Line Tools

### cURL

- Transfers files via HTTP, HTTPS, FTP, SCP, etc.
    
- Shows **page source** in terminal.
    

bash

```
curl http://localhost
```

- Output = raw HTML source.
    

### Wget

- Downloads content & saves locally.
    

bash

```
wget http://localhost
```

**Output:** saves `index.html`.

## 🐍 Python Web Server

- Lightweight alternative for quick hosting.
    
- Start server in current directory:
    

bash

```
python3 -m http.server
```

- Default port: 8000 → access via `http://localhost:8000`.
    
- Custom port:
    

bash

```
python3 -m http.server 443
```

- Custom directory:
    

bash

```
python3 -m http.server --directory /home/user/files
```

**Logs show requests** (GET requests for HTML, CSS, images).

## 🧠 Key Takeaways

- **Apache**: robust, modular, supports static/dynamic content.
    
- **cURL/Wget**: command-line tools for fetching & testing web content.
    
- **Python server**: quick file hosting, useful for penetration testing.
